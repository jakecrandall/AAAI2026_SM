# AAAI2026_SM

M. Richards, D. Cowser, D. Nielson, and J. W. Crandall
Toward Simulating Networked Societies with Formal Institutions Using AI Agents
Proceedings of the 40th Conference on Artificial Intelligence (AI for Social Impact Track) (AAAI 2026), Singapore, 2026
